For making this project we need:
 - arduino uno
 - breadbord
 - 6 resistors
 - 2 RGB LED or 6 simple LEDs (2 - red, 2-green, 2-blue) or other colors
 - bluetooth module HC-05 or HC-06

 set pins on arduino:
  2 pin - first red led
  3 pin - first green led
  4 pin - first blue led
  5 pin - second red led
  6 pin - second green led
  7 pin - second blue led
